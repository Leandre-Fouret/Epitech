# 🤖 RobotFactory - Une Partie du Projet Corewar

RobotFactory est la première étape d'un projet plus vaste appelé **Corewar**. 🛠️

## 📄 Description
Ce projet consiste à compiler des programmes appelés **Champions**. Ces programmes sont écrits dans un langage de type Assembleur simplifié, souvent appelé **Redcode** (extension `.s`).

RobotFactory se charge de compiler ces programmes tout en gérant les différents cas d'erreurs. Cela inclut :

- ✅ Reconnaître chaque **instruction**.
- 📥 Identifier et récupérer les **arguments** selon l'instruction.
- 🔍 Gérer les **types d'arguments**.
- 🧩 Effectuer la compilation en respectant les spécificités des instructions (comme l'introduction au **coding byte**).

## 🛠️ Fonctionnalités
- Le programme génère un fichier compilé avec l'extension `.cor`. 📂
- Ce fichier compilé peut ensuite être utilisé dans le projet **Corewar** pour faire s'affronter les Champions entre eux dans une arène virtuelle. ⚔️

## 🚀 Objectif
L'objectif final est de permettre aux Champions compilés de s'affronter dans le projet **Corewar**, simulant ainsi une bataille entre programmes.
